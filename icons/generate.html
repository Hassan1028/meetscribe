MeetScribe Icon Generator

Click each download link to save the PNG. Place all files in the icons/ folder.

After downloading, add this to manifest.json inside the root object:

"icons": { "16": "icons/icon-16.png", "32": "icons/icon-32.png", "48": "icons/icon-48.png", "128": "icons/icon-128.png" }

Also add inside the "action" block:

"default_icon": { "16": "icons/icon-16.png", "32": "icons/icon-32.png", "48": "icons/icon-48.png", "128": "icons/icon-128.png" }
